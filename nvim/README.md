## How to install
1. Copy this file in `.config`.
2. If you using vim, rename this folder to `.vim`.
3. If you using vim, rename `init.vim` to `.vimrc`, `_vimrc`, or `vimrc`
4. Execute `:PlugInstall`
5. Execute `:CocInstall <Extension>`
